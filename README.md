# 🎭 About The Project

> **Animated Spider Clock** is a premium web experience that reimagines the classic analog clock through the mysterious beauty of a living spider.

Inspired by **Japanese Manga**, **Cyberpunk aesthetics**, and **Glassmorphism UI**, this project transforms time into an immersive visual masterpiece. Rather than using ordinary clock hands, a highly animated spider uses its legs to indicate the **hour**, **minute**, and **second** in real time.

Every detail—from glowing neon effects and floating spider webs to cinematic particle systems and smooth CSS animations—has been carefully crafted to deliver an unforgettable experience.

Whether you're a web developer, UI/UX enthusiast, or animation lover, this project showcases how creativity and modern front-end technologies can turn a simple clock into a work of digital art.

🎨 **Pure Front-End** – Built without frameworks using only HTML, CSS, and Vanilla JavaScript.
